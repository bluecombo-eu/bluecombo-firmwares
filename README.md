## Repository structure

* [/firmwares/](/firmwares/) - Contains the firmware files for each device.
* [/scripts/](/scripts/) - Contains scripts used for repository maintenance.
* [/firmwares.json](/firmwares.json) - A JSON file that provides a structured
    representation of the firmware files in the repository. It includes
    information about each firmware such as its name, version, size, hash,
    and the path to the corresponding file in the `/firmwares/` directory.
* [/LICENSE.txt](/LICENSE.txt) - The EULA governing firmware usage.
